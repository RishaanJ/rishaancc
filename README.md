# rishaan.cc

My personal portfolio — a minimal, dark-mode-first website built with Next.js, Tailwind CSS, and Framer Motion.

**[rishaan.cc →](https://rishaan.cc)**

---

## Stack

- **Framework** — [Next.js 16](https://nextjs.org) (App Router)
- **Styling** — [Tailwind CSS v4](https://tailwindcss.com)
- **Animations** — [Framer Motion](https://www.framer.com/motion/)
- **Components** — [shadcn/ui](https://ui.shadcn.com) + [Radix UI](https://www.radix-ui.com)
- **Typography** — Geist Sans / Mono, Apple Garamond Light
- **Language** — TypeScript


## Project Structure

```
app/
├── page.tsx          # Home page
├── layout.tsx        # Root layout & fonts
├── globals.css       # Global styles & Tailwind config
components/
├── ProjectCard.tsx   # Project showcase card
├── ui/
│   ├── bento-grid.tsx    # Bento grid layout
│   ├── button.tsx        # Button component
│   └── smooth-cursor.tsx # Custom cursor animation
lib/
└── utils.ts          # Utility functions
public/               # Static assets
```
